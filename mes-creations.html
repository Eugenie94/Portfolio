Mes site web

Employeurs

J'ai travaillé en freelance pour l'entreprise OG Multi Services de Février à Mars 2020. J'ai particulièrement apprécié de travailler sur ce projet web car l'entreprise m'a donné carte blanche pour réaliser leur site web.

J'ai réalisé le site web Todolist dans le cadre d'un test en vue d'un entretien d'embauche. J'ai particulièrement aimé ce projet web car il y avait du challenge, celui de produire la meilleure réalisation afin d'avoir une chance d'être prise pour l'entretien.
De plus cette Todolist me sert pour la réalisation de mes projets web car elle m'aide à m'organiser dans ma méthodologie.
